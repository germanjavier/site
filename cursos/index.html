Curso: Mi Primera Web

100% aplicado y enfocado en resultados: Te vas del curso con tu propia página web funcionando en internet, creada con tus propias manos. No te llevás un cuaderno lleno de notas, te llevás un proyecto real en la nube.

Modalidad y Detalles

  • Duración: 7 Clases de 2.5 Horas.
  • Inversión: Pago único. Acceso a las 17 horas y media de clases en vivo por Zoom.
  • Metodología: 45 min concepto visual clave, 60 min código en vivo, 45 min consultas y feedback.
  • Requisitos: Ninguno, diseñado para cualquier edad sin conocimiento previo (14+ años).

¿Qué vas a aprender?

En estas 7 clases no vas a venir a memorizar teoría aburrida ni códigos gigantes. Vas a aprender a resolver un problema real utilizando la tecnología.

Desde la primera clase vas a elegir una única actividad principal según tu propio objetivo, y todo lo que aprendas se aplicará directamente a ella:

  • ■ Si querés mostrar tu trabajo al mundo: crear tu Portafolio o Web Personal.
  • ■ Si tenés un emprendimiento: diseñar la Web para tu Negocio.
  • ■ Si querés estudiar sistemas: entender las bases para entrar con ventaja a la unversidad.

El temario práctico incluye:

  • HTML5: Cómo armar el "esqueleto" y la estructura de forma moderna.
  • CSS3: Diseño y flexbox para lograr un aspecto profesional.
  • JavaScript: Interactividad (clics, formularios, lógica).
  • IA Aplicada: Usar IA (ChatGPT/Claude) como tu asistente de programación seguro.
  • Tu web en la Nube: Servidores reales (Netlify/Vercel) y dominios (.com).

Requisitos Clase 1

En los primeros minutos de la Clase 1, ocuparemos:

  • Editor: Visual Studio Code (Instalado).
  • Navegador: Google Chrome o Firefox (Actualizados).
  • Cuentas gratuitas: Registro en ChatGPT (u otra IA) y Netlify o Vercel.

Programa de Clases

Clase 1: El Plano de la Casa (Herramientas y Maquetado HTML)

Enfoque: Entender cómo funciona la web y estructurar el contenido de su proyecto.

  • ■ ¿Cómo viaja una web por internet? HTML es el esqueleto, CSS la ropa, JS los músculos.
  • ■ Herramientas a instalar (En vivo): Visual Studio Code + Live Server.
  • ■ Estructura base moderna y etiquetas semánticas.
  • Actividad Aplicada: Cada alumno crea el esqueleto de su página principal.

Clase 2: Pintando la Web (Diseño y Estilos con CSS)

Enfoque: Que dejen de ver texto plano y su proyecto empiece a verse profesional.

  • ■ Conectar CSS con HTML y la anatomía de una regla CSS.
  • ■ El modelo de caja y Flexbox básico (la herramienta moderna indispensable).
  • Actividad Aplicada: Darle estilo a la estructura de la Clase 1, cambiar fondos y fuentes.

Clase 3: Darle Vida a la Web (Interactividad con JavaScript)

Enfoque: Perderle el miedo al código y hacer que la web «reaccione».

  • ■ Variables, Funciones y manipulación del DOM.
  • ■ Escuchar un clic: addEventListener('click', ...).
  • Actividad Aplicada: Añadir interacciones reales al proyecto (un "ver más", precios calculados o validadores básicos).

Clase 4: La Gran Integración (Uniendo los Tres Mundos)

Enfoque: Consolidar lo aprendido y resolver el problema principal del proyecto fluídamente.

  • ■ Flujo de trabajo: Idea → HTML → CSS → JS.
  • ■ Creación de una sección compleja como una Galería o Contacto.
  • Actividad Aplicada: Integración total de la web principal y resolución de errores en vivo.

Clase 5: El Copiloto Inteligente (Uso Seguro y Rendimiento con IA)

Enfoque: Enseñarte a usar IA como un asistente, no como un reemplazo, cuidando tu seguridad.

  • ■ Cómo pedirle código a una IA (Estructura de un prompt).
  • ■ El peligro del "Copiar y Pegar" y buenas prácticas de seguridad de Datos.
  • Actividad Aplicada: Los alumnos usan la IA para agregar una funcionalidad compleja (ej. Carrusel o Modo oscuro).

Clase 6: Lanzamiento al Mundo (Nube, Dominios y Despliegue)

Enfoque: Que el proyecto deje de vivir en la PC y cualquiera pueda verlo online.

  • ■ Subida fácil en Vercel o Netlify (arrastrando la carpeta).
  • ■ Qué es un Hosting, Dominio y qué es el certificado SSL.
  • Actividad Aplicada: Cada participante sube su sitio a internet en vivo y se comparte en chat.

Clase 7: El Mundo Laboral y Cierre de Proyectos

Enfoque: Qué buscan las empresas enfocadas en resolución de problemas.

  • ■ Habilidades blandas: comunicar, ser proactivo e investigar.
  • Showcase: Cada alumno presenta por 3 minutos su proyecto final operando en la nube.
  • Retroalimentación de pares, consultas finales y entrega de Diploma digital.
